Frequently Asked Questions

How do I create a group?

+

To create a group go to the sidebar and click on the "Create Group". You can name your group and generate a unique join code that you can share it with others.

Can I be part of multiple groups?

+

Yes, you can join multiple groups using their unique join codes.

How do I add tasks or events?

+

Within a group, use the calendar interface to add tasks or events. Click on a date, enter details, and click save.

How to create an event?

+

To create an event, go to the sidebar and click on the "Create Event", select a date and time, and enter the event details. Click "Save" to create the event and it will show in the chat so that all people in the group can vote if they are available or not if the majority of people are free the event is going to be created.
